Hardcore Reclamation


Non-exhaustive list of changes from base Reclamation:
* **Mystical Agriculture removed - resource generation is now focused around Theurgy, Complicated Bees, and others**
* New mods
    * Cold Sweat (temperature)
    * Diet (nutritional groups)
        * Every single food has been rebalanced to incentivize crafting higher-tier meals
    * Thirst Was Taken (thirst)
        * All water in the world starts out as dirty
        * Custom camel packs have been added for handsfree drinking
        * **Water cannot be turned into infinite pools**
        * Cooking for Blockheads sink has been nerfed
* Automation
    * Embers fluid and item pipes no longer require lead
    * RF generation is more important earlier
    * Basic Mekanism logistics are accessible earlier
    * Enchanted's cauldron and kettle can be refilled automatically
    * Botania redstone spreader craftable earlier
* Theurgy
    * Digestion is faster
    * **Automation is much less convoluted**
        * Machines only allow extraction of outputs
        * Digestion and Fermentation Vats output a comparator signal when there's a valid recipe in them
        * The comparator signal strength scales with how full the output inventory is
        * Many processes can be done in Mekanism machines
* Complicated Bees
    * Bees are faster to get into
    * Bee quest tree has been expanded and clarified
    * Mutations have been overhauled to provide more sensible tiering
    * Several new bees have been addded
* Blood Magic
    * Slates are harder to make
    * Meteors are smaller and cost more to summon
    * Demon Realm is harder
* Miscellaneous Tweaks
    * Snowballs no longer produce aura at the Shooting Mark
    * Spruce trees no longer convert dried earth to podzol
* Many more biome globes have been added
* Globesome has a secret use
* Many tweaks and reshuffles to recipes
* Many, many bugfixes
* ~50 new quests explaining related changes and new survival mechanics
